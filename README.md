# RN101-class

Example for React Native 101 class. Each commit represents each step.

## How to run

### Clone the repository

```bash
$ git clone https://github.com/cwdoh/RN101-class.git
```

### Install whole dependencies

```bash
$ cd RN101-class
$ npm install
```

### Execute

```bash
$ npm run start
```
